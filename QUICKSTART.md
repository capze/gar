# 🚀 垃圾分類辨識系統 - 快速開始指南

## ⚡ 立即開始

### 1. 啟動本地靜態伺服器
- 建議使用 `npx http-server -p 8000` 或 `python3 -m http.server 8000`

### 2. 訪問應用
- **電腦**: http://localhost:8000
- **手機** (同一網路): http://<電腦IP>:8000

### 2. 操作步驟

```
開啟相機 → 對準垃圾 → 拍照 → 查看結果 → 重新拍照
```

### 3. 快速提示

| 步驟 | 操作 |
|-----|------|
| 📷 | 點擊「開啟相機」|
| 📸 | 對準要分類的垃圾 |
| 🎯 | 點擊「拍照」|
| 📊 | 查看辨識結果 |
| 🔄 | 點擊「重新拍照」重試 |

## 📊 結果解讀

### 信心度指標

- **🟢 綠色** = 高信心度 (≥70%) → 結果可信
- **🟠 橙色** = 中等信心度 (40-70%) → 相對可信
- **🔴 紅色** = 低信心度 (<40%) → 建議重新拍照

### 垃圾分類

| 分類 | 說明 | 範例 |
|-----|------|------|
| **Trash** ✨ | 一般廢棄物 | 紙巾、包裝紙 |
| **Shoes** 👟 | 鞋類 | 破舊鞋、拖鞋 |
| **Plastic** 🥤 | 塑膠製品 | 塑膠瓶、塑膠袋 |
| **Glass** 🍾 | 玻璃製品 | 玻璃瓶、玻璃杯 |

## 💡 使用技巧

### 提高辨識準確度

✅ **要做的事**:
- ☀️ 確保光線充足
- 📐 將垃圾放在畫面中央
- 🔍 讓垃圾佔據大部分畫面
- 📹 保持相機穩定
- 🔄 從不同角度多次嘗試

❌ **不要做的事**:
- 🌑 在昏暗的環境中
- 🚫 垃圾太小或不清晰
- 👆 經常移動相機
- 🔄 多個物品混在一起
- ⚡ 逆光拍攝

- ✅ 有攝像頭的設備

## 🎯 常見場景

### 場景 1: 我要丟掉一個塑膠瓶

1. 點擊「開啟相機」
2. 對準塑膠瓶，確保整個瓶子可見
3. 點擊「拍照」
4. 系統會給出分類建議 → **Plastic (塑膠)**

### 場景 2: 我不確定某個物品是什麼

1. 從多個角度拍照
2. 查看各個分類的信心度
3. 信心度最高的分類就是最可能的答案
4. 如果都不確定（都是紅色），可能不在訓練範圍内

### 場景 3: 辨識失敗

如果多次拍照都顯示红色 (信心度低):

1. 確保光線充足 ☀️
2. 調整角度 🔄
3. 物體要清晰可見 🔍
4. 確保物體完全在預覽框內 📐

## 🛠️ 故障排除

### ❓ 相機無法打開

**症狀**: 點擊「開啟相機」後沒有反應

**解決方案**:
1. 檢查瀏覽器是否要求權限
2. 點擊允許訪問相機
3. 檢查個人隱私設定
4. 嘗試其他瀏覽器
5. 確認設備確實有相機

### ❓ 模型加載慢

**症狀**: 應用加載時間長

**解決方案**:
1. 這是正常的 (首次加載需要下載 ~2MB 模型)
2. 刷新頁面重試
3. 檢查網速
4. 關閉其他標籤頁

### ❓ 辨識結果不準確

**症狀**: 總是辨識錯誤

**解決方案**:
1. 改善光線環境
2. 調整物體位置
3. 物體應該佔據大部分畫面
4. 多試幾次，模型有變差

### ❓ 應用運行緩慢

**症狀**: 拍照後很久才反應

**解決方案**:
1. 關閉不必要的應用
2. 重啟瀏覽器
3. 清除瀏覽器快取
4. 使用較新的瀏覽器版本

## ❓ 常見問題

**Q: 應用能離線工作嗎?**
A: 可以，首次加載完成後無需網路。

**Q: 準確率是多少?**
A: 約 85%，取決於物體清晰度和光線。

**Q: 手機會發熱嗎?**
A: 推理時可能有輕微發熱，這是正常的。

**Q: 支援多少種物品?**
A: 目前支援 4 種：垃圾、鞋類、塑膠、玻璃。

**Q: 可以訓練自己的模型嗎?**
A: 可以，使用 Google Teachable Machine 平台。
